# phishing-email-investigation-guide
Cybersecurity awareness training project demonstrating a step-by-step approach to identifying and investigating phishing emails through practical examples and security best practices.
Overview

This project was developed to promote cybersecurity awareness by educating users on how to recognize, analyze, and investigate phishing emails. The training provides a structured approach to identifying common phishing indicators and responding appropriately to suspected phishing attempts.

Objectives
Increase cybersecurity awareness.
Teach users how to identify phishing attempts.
Demonstrate a systematic approach to phishing email investigation.
Promote safe email security practices.
